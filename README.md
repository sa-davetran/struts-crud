## Developer Setup

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Apache Maven
- Git

### Installation
1. Clone the repository:
	```sh
	git clone https://github.com/sa-davetran/struts-crud.git
	cd struts-crud
	```
2. Build the project:
	```sh
	mvn clean install
	```

### Running the Application
1. Start the application using Maven:
	```sh
	mvn jetty:run
	```
2. Open your browser and go to `http://localhost:8080/crud`.
